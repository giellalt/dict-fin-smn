# Dictionary for fin — smn

This repository contains source files for a dictionary from Finnish to Inari Saami. The content is licensed under the CC-BY-4.0 license.

Many of the dictionaries are published on [sátni.org](https://sátni.org) and [NDS](https://sanit.oahpa.no).
